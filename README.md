Login & Registration Form
Description

This project is a simple Login and Registration form built using HTML and CSS. It provides a clean user interface that allows users to switch between login and registration forms without using JavaScript.

Features
Login form interface
Registration form interface
Toggle between login and register using CSS only
Simple and clean UI design
Responsive centered layout
Technologies Used
HTML
CSS
Project Structure
index.html
style.css
How to Run
Download or clone the project
Open the folder
Open index.html in any web browser

No installation required.

How it Works

The form switching is controlled using a hidden checkbox in HTML. CSS :checked selector is used to show and hide the login and registration forms.
